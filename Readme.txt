Կոդն աշխատացնելու համար անհրաժեշտ է այն գրել տեքստային ֆայլի մեջ:
CPP ֆայլը աշխատեցնելուց հետո՝ հայտնվում է "Enter the name of the file:"  , որից հետո պետք է մուտքագրել տեքստային ֆայլի միայն անվանումը ( առանց .txt   ):

 -    Լեզուն ունի 3 տիպ՝ ամբողջ և լողացող թվերի համար(num), տրամաբանակն(logic) և սիմվոլների համար նախատեսված տիպ(str), 
   որը կարող է ունենալ թվեր, տառեր և սիմվոլներ: 

-    Հայտարարությունը սկսվում է փոփոխականի տիպով, փոփոխականի անունով և/կամ արժեքով և/կամ արտահայտությամբ:

-    Փոփոխականները հայտարարվում են միայն փոքրատառ լատինատառերով

-    Բոլոր տոկենների միջև պարտադիր բացատ և տողը ավարտվում է պարտադիր ";" -ով (բացի "newline" -ից և "endif" -ից )

-    (num) տիպի հետ կարող ենք կատարել գումարման, հանման, բազմապատկման և բաժանման գործողությունները
      նույն արտահայտության մեջ մեկից ավել գործողություն կատարել չենք կարող:
      Օրինակ՝  
	a = a + b ;    (կարող ենք)
	a = a + b + c ;   (չենք կարող)	
		
-    Տպել գործողությունը կատարվում է ("print"  բացատ   "->"   բացատ   "արտահայտություն"   բացատ ";" ) 
     "print" -ով ամեն անգամ կարող ենք տպել մեկ արտահայտության արդյունք
     Օրինակ՝  
	print -> a + b ;    (կարող ենք)
	print -> a + b -> a - b ;   (չենք կարող)	
     եթե ուզում ենք մեկ "print" -ով  տպել որևէ տեքստ, ապա տվյալ տեքստում չպետք է լինեն բացատներ	
Օրինակ՝  
	print -> Barev_Hayk_jan ;    (կարող ենք),
	print -> Barev ; print -> Hayk ; print -> jan ;    (կարող ենք),
	print -> Barev Hayk jan;   (չենք կարող)		


-    if-ը ունի հետևյան սինթաքսը՝ ("if"  բացատ   "->"   բացատ   "արտահայտություն"  բացատ    ";"  )    
     Եթե "->"  ի աջ մասում արդյունքը ստացվում է true, ապա կատարում է if - ի  մարմինը:  
     Եթե false է ապա endif-ով ավարտում է if-ը: endif - ը ազդարարում է if -ի  մարմինի կատարման ավատը: endif-ը վերջում   ";"   նշան չունի

 -   newline - ով անցնում ենք հաջորդ տող, վերջում   ";"   նշան չունի:

 -  բանալի բառերն են՝  "num", "str", "logic", "newline", "if", "endif", "print" :
